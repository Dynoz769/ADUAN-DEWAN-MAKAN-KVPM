# Sistem Aduan Dewan Makan

Ini adalah sistem aduan untuk pelajar di Dewan Makan, yang membolehkan pelajar untuk menghantar aduan berkaitan dengan makanan yang disediakan.

## Ciri-ciri:
- Pelajar boleh menghantar aduan mengenai menu makanan.
- Admin boleh menguruskan aduan dan memantau status aduan.
- Sistem ini menggunakan HTML, CSS dan JavaScript.

## Cara Penggunaan:
1. Muat naik fail ini ke GitHub dan buka URL untuk melihat sistem berfungsi.
2. Log masuk untuk hantar aduan dan lihat senarai aduan yang diterima.

## Teknologi:
- HTML, CSS
- JavaScript
